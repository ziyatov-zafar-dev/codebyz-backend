# CodeByZ Auth Backend (No Lombok) - v2

✅ Roles: STUDENT / TEACHER / ADMIN  
✅ Sign-up faqat STUDENT yoki TEACHER (Google bilan sign-up yo'q)

✅ Admin approval flow (CHECKING -> CONFIRMED/CANCEL)
